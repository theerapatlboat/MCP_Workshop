

**REQUIREMENT**

**Facebook Page  Sales Closing Chatbot**

Chatbot ปิดการขายเชื่อมต่อ Facebook Page

|   |
| :---- |

→  ตอบแชทลูกค้าผ่าน Facebook Page ได้อัตโนมัติ

→  รับข้อมูลจากลูกค้าได้ 3 รูปแบบ: ข้อความ, รูปภาพ, เสียง

→  ดึงข้อมูลสินค้าและรูปภาพมาตอบลูกค้าได้ถูกต้อง

→  จำบทสนทนาลูกค้าคนเดิมได้

→  สร้างออเดอร์เมื่อลูกค้ายืนยัน

→  ตรวจสอบความถูกต้องของที่อยู่

→  ตอบข้อมูลสินค้าจากข้อมูลที่กำหนดให้

→  ไม่ตอบนอกขอบเขต ไม่ให้ข้อมูลเท็จ ไม่ให้ส่วนลดเกินกำหนด

→  Guardrail ป้องกันข้อความที่ไม่ต้องการ

→  เดินตาม Sale Script ตั้งแต่ทักทายจนถึงปิดการขาย

→  ส่งต่อให้พนักงานจริงได้เมื่อ AI จัดการไม่ได้
